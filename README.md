# Desert Sky WorkApp Demo

This is a standalone demo app for presenting the Desert Sky workflow concept.

It is inspired by the Job Well Done product structure, but it is built around the WorkFlowOS / Desert Sky workflow:

- Field: mobile employee and crew job view
- Dox: digital forms and job documents
- Ping: owner, customer, and manager notifications
- Log: activity feed and change tracking
- Inventory: have, need, low, missing
- Spanish-friendly employee actions

## Run Locally

From this folder:

```bash
npm.cmd run dev
```

Then open:

```text
http://localhost:5180
```

## Build

```bash
npm.cmd run build
```

The production build outputs to:

```text
desertskyworkapp/dist
```

## Demo Talk Track

This app shows how Desert Sky can give employees one simple workspace for jobs, field updates, forms, inventory, notifications, and activity tracking.

The goal is not to copy another product. The goal is to build a simple workflow app that fits Desert Sky's daily operation and can later become a reusable WorkFlowOS client template.
